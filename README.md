## EmployeesApp

Aplikasi web sederhana untuk mengelola **data karyawan** menggunakan **ASP.NET Core MVC**.

### Teknologi

- .NET 10 (ASP.NET Core MVC)
- C#
- Entity Framework Core InMemory
- Bootstrap 5
- jQuery

### Fitur

- Monitoring data karyawan (tabel + filter NIK & Nama).
- Tambah data karyawan.
- Edit data karyawan.
- Detail data (read-only).
- Hapus data dengan konfirmasi alert.

### Installation

1. Clone repository:

   ```bash
   git clone https://github.com/shofiyatunnisa9/Employees-App.git
   cd employees-app
   ```

2. Jalankan aplikasi:

   ```bash
   dotnet run
   ```
